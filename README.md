## 1. 프로젝트 소개

##### :one:  계획의 현재 달성률을 progress bar(Bootstrap)를 이용하여 시각화하여 나타낸다.
##### :two:  [My Plan Board]에서는 진행중인 모든 계획의 현황을 일괄적으로 보여준다.
##### :three:  달성률은 유저가 계획 입력시 작성한 [중간 목표]와 [완수율] 수치를 통해 계산된다.
##### :four:  유저는 자신의 활동 관리를 통해 달성률을 업데이트 할 수 있다.
